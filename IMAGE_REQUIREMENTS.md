# Image Requirements for SOLTAKO-Style Homepage

This document lists all the images you need to prepare for the website to match the SOLTAKO design.

## 1. Hero Carousel Images (3 images)
**Location:** Top of homepage, full-width carousel
**Dimensions:** 1920x1080px (or similar wide format, minimum 1600px width)
**Format:** JPG or WebP, optimized for web
**Content:**
- **Hero Image 1:** Lifestyle shot - Elegant room/kitchen with ceramic tiles prominently displayed. Should convey luxury and Mediterranean aesthetic.
- **Hero Image 2:** Product showcase - Beautiful arrangement of ceramic tiles in a modern setting
- **Hero Image 3:** Lifestyle shot - Tiles in use in a bathroom or kitchen, showing the product in context

**Note:** Each image should have space for text overlay on the left side (title, subtitle, button)

## Image Optimization Guidelines

1. **File Size:** Keep images under 500KB each (use compression tools)
2. **Format:** 
   - Photos: JPG (quality 80-85%)
   - Logos/Icons: PNG or SVG
   - Product images: WebP if possible
3. **Naming Convention:** Use descriptive names like:
   - `hero-carousel-1.jpg`
   - `feature-circle-geschenk.jpg`
   - `customer-review-1.jpg`
   - etc.
4. **Alt Text:** We'll add descriptive alt text for accessibility
